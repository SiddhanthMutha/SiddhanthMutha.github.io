◈ Projects ⌄
FuseRAG

A production-minded RAG system for chatting with documents, code, markdown, and web content without treating retrieval like an afterthought. FuseRAG fuses semantic vector search with BM25 keyword matching via Reciprocal Rank Fusion, then applies cross-encoder reranking to surface the most relevant chunks before generation.

Built in an evaluation-first mindset with metrics including Precision@k, Recall@k, MRR, nDCG, Faithfulness, and Correctness. Features a FastAPI backend, Pinecone-compatible vector store, PostgreSQL/pgvector metadata storage, and a lightweight web UI with Jinja2 and HTMX for end-to-end document ingestion, querying, and evaluation.

RAG Hybrid Retrieval BM25 Reciprocal Rank Fusion Cross-Encoder FastAPI Pinecone PostgreSQL HTMX Evaluation
ResearchFlow

A small but real LangGraph research system with four specialized agents working together: a Coordinator that turns raw prompts into cleaner research tasks, a Search Agent that finds relevant web results, a Reader Agent that fetches and extracts the useful signal from pages, and a Writer Agent that synthesizes everything into a final answer with sources.

Features a polished static frontend served directly from FastAPI, live WebSocket progress updates while each agent works, and an SQLite history layer so research runs don't disappear after one request. Built to be a clean end-to-end product surface: orchestrated agents, API, persistence, and a frontend that makes the orchestration visible.

LangGraph Multi-Agent FastAPI WebSocket SQLite Tavily Frontend
